Controle e Logs do Robô

Uma aplicação web desenvolvida com Flask e TinyDB para controlar um robô e registrar seus logs de atividades.

Componentes da Aplicação

Esta aplicação utiliza Flask para a criação da interface web e TinyDB para armazenamento de logs.

Funcionalidades Principais

Inclui controle do robô via interface web e visualização de logs para monitoramento de atividades.

Vantagens

Oferece simplicidade, leveza, flexibilidade, e independência de plataforma.
